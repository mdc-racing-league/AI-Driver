# IBM AI Racing League – TORCS Driver

## Overview
This project contains our AI driver developed for the IBM AI Racing League using TORCS (The Open Racing Car Simulator).

The goal is to build an autonomous racing agent that can navigate a track using real-time sensor data and make driving decisions such as steering, acceleration, and braking.

---

## What is the AI Racing League?

The AI Racing League is an opportunity for students to:
- Showcase AI development skills
- Work as a team to solve real-world problems
- Communicate technical ideas effectively
- Build practical experience using industry tools

Teams (3–5 members) develop an AI race car using:
- TORCS
- Python
- IBM Granite
- IBM SkillsBuild
